About the Project

Purpose of the System

ZenTasker is a web-based task management system designed to help users organize, track, and manage their daily tasks efficiently. As tasks continue to increase, especially among students and employees, traditional methods like notepads become ineffective, making ZenTasker a reliable digital solution.

The application allows users to create, view, edit, filter, and manage tasks within a well-organized and user-friendly interface. Tasks are categorized into views such as All Tasks, Today’s Tasks, and Important Tasks, helping users focus on urgent responsibilities.

By utilizing structured data handling and efficient task management logic, the system enhances productivity, reduces missed deadlines, and promotes better time management. Additionally, the system utilizes light and dark mode to enhance user experience by providing visual comfort.

Task Management

  • Add tasks with title, description, date, and time
  • Edit and delete existing tasks
  • Mark tasks as completed
  • Mark tasks as important

Filtering & Sorting

  • Search by title or description
  • Filter by Priority, Status, or Date
  • Sort by Due Date, Creation Date, or Title
  • Ascending or Descending order

Statistics & UI

  • Real-time dashboard (Total, Pending, Overdue)
  • Dark Mode toggle for visual flexibility
  • Expandable full-screen description editor
  • Automatic priority assignment
